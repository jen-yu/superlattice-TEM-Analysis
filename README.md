# superlattice-TEM-analysis

Key parameters
1. min_feature_size 
2. morphology
3. small
4. thershold in deleting voronoi cell by area 

Steps
1. Read file 
2. Get image scale 
3. Get particle centers/radii and mask 
4. Voronoi cells
5. MSM structure to get the assembly orientation
6. plot symmetry map
